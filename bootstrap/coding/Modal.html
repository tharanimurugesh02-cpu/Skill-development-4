Please review our usage policies thoroughly before proceeding:

1. Smooth Zoom Animation: Notice how the modal scales smoothly up from the center when it opens, rather than dropping abruptly from the top of the viewport. We achieved this using a couple of custom CSS rules on the .zoom-anim class wrapper.

2. Centered Alignment: Thanks to the modal-dialog-centered utility utility, the frame scales dynamically to sit precisely dead-center on desktops, laptops, tablets, and smartphones effortlessly.

3. Scrollable Layout: We added modal-dialog-scrollable. This pins the header and footer firmly in place. Go ahead and scroll up or down inside this panel; only the inner core text area moves, creating a clean mobile-app feel.
